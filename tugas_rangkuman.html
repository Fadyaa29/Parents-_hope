JavaScript Summary

JS Introduction

JavaScript is a programming language for creating interactive and dynamic web content.

JS Where To

JavaScript can be placed inside HTML using the <script> tag or linked as external files with the .js extension.

JS Output

Methods for outputting data include:

  • document.write()
  • console.log()
  • alert()
  • Modifying HTML elements using innerHTML.
JS Statements

Statements are JavaScript instructions executed by the browser, such as variable declarations or function calls.

JS Syntax

JavaScript is case-sensitive. Code blocks are defined with curly braces {}, and statements end with a semicolon ;.

JS Comments

Comments are used to describe code:

  • Single-line: //
  • Multi-line: /* */
JS Variables

Variables are used to store data. Declared using var, let, or const.

JS Let

The let keyword declares block-scoped variables.

JS Const

The const keyword declares variables with values that cannot be changed.

JS Operators

Operators are used to perform operations on variables and values:

  • Arithmetic: +, -, *, /
  • Logical: &&, ||, !
JS Arithmetic

Arithmetic operators include:

  • Addition: +
  • Subtraction: -
  • Multiplication: *
  • Division: /
JS Assignment

Assignment operators, like =, are used to assign values to variables.

JS Data Types

JavaScript data types include:

  • Primitive: String, Number, Boolean, Null, Undefined
  • Object types: Array, Function, Object
JS Functions

Functions are blocks of code designed to perform a specific task:

function greet(name) { return "Hello " + name; }
JS Objects

Objects are collections of properties and values:

let car = { brand: "Toyota", model: "Corolla", year: 2022 };

JavaScript Summary

JS Object Properties

Properti objek adalah atribut yang menyimpan nilai. Contoh:

const car = { brand: "Toyota", year: 2020 };
console.log(car.brand); // Output: Toyota

JS Object Methods

Metode adalah fungsi yang terkait dengan objek. Contoh:

const person = {
    name: "Alice",
    greet: function() {
        return Hello, ${this.name};
    }
};
console.log(person.greet()); // Output: Hello, Alice

JS Object Display

Objek dapat ditampilkan menggunakan console.log() atau JSON.stringify().

const obj = { key: "value" };
console.log(JSON.stringify(obj)); // Output: {"key":"value"}

JS Object Constructors

Konstruktor adalah fungsi untuk membuat objek baru.

function Car(brand, year) {
    this.brand = brand;
    this.year = year;
}
const car = new Car("Toyota", 2020);
console.log(car.brand); // Output: Toyota

JS Events

Event adalah aksi yang dapat diproses oleh JavaScript, seperti klik tombol.

document.getElementById("myButton").addEventListener("click", function() {
    alert("Button clicked!");
});

JS Strings

String digunakan untuk menyimpan teks.

let greeting = "Hello, World!";
console.log(greeting.length); // Output: 13

JS String Methods

Metode string digunakan untuk memanipulasi teks:

  • toUpperCase(): Mengubah menjadi huruf kapital.
  • toLowerCase(): Mengubah menjadi huruf kecil.
  • trim(): Menghapus spasi di awal/akhir string.

JS String Search

Mencari teks dalam string:

let text = "Hello, JavaScript!";
console.log(text.includes("JavaScript")); // Output: true

JS String Templates

Template literal mempermudah penggabungan string:

let name = "Alice";
console.log(Hello, ${name}!); // Output: Hello, Alice!

JS Numbers

Number adalah tipe data untuk angka.

let x = 10, y = 20;
console.log(x + y); // Output: 30

JS BigInt

BigInt digunakan untuk angka yang sangat besar.

let big = 123456789012345678901234567890n;
console.log(big + 1n); // Output: 123456789012345678901234567891n

JS Number Methods

Metode untuk angka:

  • toFixed(): Membulatkan angka ke jumlah desimal tertentu.
  • toString(): Mengubah angka menjadi string.
  • parseInt(): Mengonversi string menjadi bilangan bulat.

JS Number Properties

Properti angka:

  • MAX_VALUE: Nilai maksimal angka.
  • MIN_VALUE: Nilai minimal angka.
  • NaN: Not a Number.

JS Arrays

Array digunakan untuk menyimpan beberapa nilai dalam satu variabel.

let fruits = ["Apple", "Banana", "Cherry"];
console.log(fruits[1]); // Output: Banana

JS Array Methods

Metode untuk array:

  • push(): Menambahkan elemen di akhir array.
  • pop(): Menghapus elemen terakhir.
  • shift(): Menghapus elemen pertama.
  • unshift(): Menambahkan elemen di awal array.
JavaScript Concepts

JS Array Search

Mencari elemen dalam array menggunakan metode seperti indexOf, find.

const arr = ['apple', 'banana'];
arr.indexOf('banana'); // Output: 1
Array Sort

Mengurutkan array menggunakan sort().

const nums = [3, 1, 2];
nums.sort(); // [1, 2, 3]
JS Array Iteration

Iterasi array dengan metode seperti forEach, map, atau filter.

const nums = [1, 2, 3];
nums.forEach(num => console.log(num));
JS Array Const

Array dengan const dapat dimodifikasi tetapi tidak bisa diganti sepenuhnya.

const arr = [1, 2];
arr[0] = 3; // OK
arr = [4, 5]; // Error
JS Dates

Gunakan Date() untuk menangani tanggal dan waktu.

const now = new Date();
console.log(now);
JS Date Formats

Format tanggal seperti YYYY-MM-DD atau ISO.

const date = new Date();
console.log(date.toISOString());
JS Date Methods

Gunakan metode seperti getFullYear(), getMonth().

const date = new Date();
console.log(date.getFullYear());
JS Date Set Methods

Atur tanggal dengan metode seperti setFullYear().

const date = new Date();
date.setFullYear(2025);
JS Math

Operasi matematika seperti Math.sqrt() atau Math.pow().

console.log(Math.sqrt(16));
JS Random

Generate angka acak dengan Math.random().

console.log(Math.random());
JS Booleans

Boolean hanya memiliki dua nilai: true atau false.

const isAvailable = true;
JS Comparisons

Operator perbandingan seperti == atau ===.

console.log(5 === '5'); // false
JS If Else

Gunakan if, else if, dan else untuk logika.

if (x > 0) {
  console.log('Positive');
}
JS Switch

Struktur kontrol dengan banyak kondisi.

switch(day) {
  case 1: console.log('Monday');
}
JS Loop For

Gunakan perulangan untuk iterasi kode.

for (let i = 0; i < 3; i++) {
  console.log(i);
}
JavaScript Complete Topics

JS Loop For In

Loop digunakan untuk iterasi properti enumerable dari sebuah objek.

const person = { name: "Alice", age: 25 };
for (let key in person) {
    console.log(${key}: ${person[key]});
}

JS Loop For Of

Loop digunakan untuk iterasi item dalam iterable seperti array atau string.

let fruits = ["apple", "banana"];
for (let fruit of fruits) {
    console.log(fruit);
}

JS Loop While

Loop while digunakan untuk iterasi selama kondisi tertentu benar.

let i = 0;
while (i < 5) {
    console.log(i);
    i++;
}

JS Break

Menghentikan loop sebelum selesai.

for (let i = 0; i < 10; i++) {
    if (i === 5) break;
    console.log(i);
}

JS Iterables

Objek yang bisa diiterasi menggunakan for..of, seperti array dan string.

JS Sets

Objek yang menyimpan nilai unik.

const set = new Set([1, 2, 3, 3]);
console.log(set); // Output: Set(3) {1, 2, 3}

JS Set Methods

Metode yang digunakan dengan Set: add(), delete(), has(), clear().

JS Maps

Objek pasangan kunci-nilai yang mempertahankan urutan penambahan.

const map = new Map();
map.set("key1", "value1");
console.log(map.get("key1")); // Output: value1

JS Map Methods

Metode yang digunakan dengan Map: set(), get(), delete(), has(), clear().

JS Typeof

Operator yang digunakan untuk mengecek tipe data.

console.log(typeof "hello"); // Output: string

JS Type Conversion

Mengonversi tipe data, misalnya string ke number atau sebaliknya.

let num = Number("10");
console.log(typeof num); // Output: number

JS Destructuring

Mengambil nilai dari array atau properti objek menjadi variabel individu.

const [a, b] = [1, 2];
const { name, age } = { name: "Alice", age: 25 };

JS Bitwise

Operator untuk manipulasi bit: & (AND), | (OR), ^ (XOR).

JS RegExp

Ekspresi reguler untuk mencocokkan pola teks.

let regex = /hello/i;
console.log(regex.test("Hello")); // Output: true

JS Precedence

Menentukan prioritas operator dalam ekspresi.

JavaScript Summary

JS Errors

Error seperti SyntaxError, TypeError, dan ReferenceError membantu menangkap kesalahan kode.

// Example of TypeError
let num = 5;
num.toUpperCase(); // Error: toUpperCase is not a function
JS Scope

Scope menentukan di mana variabel dapat diakses dalam kode.

let globalVar = "I'm global!";
function testScope() {
  let localVar = "I'm local!";
  console.log(globalVar); // OK
  console.log(localVar);  // OK
}
testScope();
console.log(localVar); // Error: localVar is not defined
JS Hoisting

Deklarasi diangkat ke atas scope sebelum eksekusi.

console.log(myVar); // undefined
var myVar = "Hoisted!";
function hoistTest() {
  console.log(myFunction()); // Hoisted Function!
  function myFunction() {
    return "Hoisted Function!";
  }
}
JS Strict Mode

Mendeteksi kesalahan dan mencegah penggunaan fitur JS yang tidak aman.

'use strict';
myVar = 10; // Error: myVar is not defined
JS this Keyword

this merujuk ke objek yang memanggil fungsi saat ini.

const person = {
  name: 'John',
  greet() {
    console.log(this.name); // 'John'
  }
};
person.greet();
JS Arrow Function

Fungsi ringkas, sering digunakan dengan callback.

const add = (a, b) => a + b;
console.log(add(5, 3)); // 8
JS Classes

Kelas adalah template untuk membuat objek dengan properti dan metode tertentu.

class Animal {
  constructor(name) {
    this.name = name;
  }
  speak() {
    return ${this.name} makes a noise.;
  }
}
const dog = new Animal('Dog');
console.log(dog.speak());
JS Modules

Memungkinkan membagi kode menjadi file kecil.

// module.js
export function greet(name) {
  return Hello, ${name}!;
}

// main.js
import { greet } from './module.js';
console.log(greet('World'));
JS JSON

JSON adalah format data ringan untuk mentransfer data.

const json = '{"name": "Alice", "age": 25}';
const obj = JSON.parse(json);
console.log(obj.name); // Alice
JS Debugging

Gunakan console.log atau DevTools untuk debugging.

let count = 0;
console.log('Count is:', count);
JS Style Guide

Panduan penulisan kode JavaScript yang konsisten dan mudah dibaca.

// 1. Gunakan camelCase untuk nama variabel
let firstName = "John";

// 2. Tambahkan titik koma di akhir pernyataan
let age = 25;

// 3. Gunakan single quotes untuk string
let city = 'New York';

// 4. Gunakan indentasi 2 spasi
function greet(name) {
  console.log('Hello, ' + name);
}

// 5. Hindari kode yang terlalu panjang
let message = 'This is a very long message that should be broken into smaller parts for better readability.';
JS Best Practices

Gunakan nama variabel bermakna dan hindari kebocoran global.

// Avoid global variables
let total = 100; // Good practice
JS Mistakes

Kesalahan umum meliputi penggunaan variabel tanpa deklarasi atau lupa menutup tanda kurung.

// Contoh variabel tanpa deklarasi
function calculate() {
  result = 10; // Error: variabel tanpa deklarasi
}
JS Performance

Optimalkan performa dengan meminimalkan manipulasi DOM dan menggunakan metode efisien.

// Batasi akses ke DOM
let element = document.getElementById('item');
for (let i = 0; i < 1000; i++) {
  element.style.color = 'blue';
}
JS Reserved Words

Beberapa kata kunci JavaScript dipesan dan tidak dapat digunakan sebagai nama variabel.

// Contoh penggunaan kata kunci yang dilarang
let class = "Math"; // Error
let function = "run"; // Error
